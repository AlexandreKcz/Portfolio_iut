Sae 1.03 : Installation d'un poste pour le developpement

Cette saé avait pour but de configurer des outils de travail et de développement sur sa machine personnelle
Cette SAÉ est la première liée a la compétence 'Administrer des Systèmes informatiques communicants complexes'.
Elle précède l'installation de services réseaux et permet de comprendre l'architecture de base d'un système d'exploitation, avant de le connecter a un réseau

Les productions :
  1. Installation d'outils de développement et de travail sur sa machine personnelle
  2. Installation et démonstration d'une machine virtuelle linux
  3. Rédaction d'un tutoriel décrivant l'installation d'une machine linux vierge
  4. Rapport d'autoévaluation

L’objectif est de pouvoir analyser le fonctionnement d'un système d'exploitation et d'en comprendre les différentes étapes d'installation et de configuration


Le sujet proposé etait l'installation d'une machine virtuelle linux debian
Les différents problèmes posés reposaient sur l'utilisation de la machine virtuelle et son interaction avec le système d'exploitation primaire


Ce projet a été réalisé en équipe imposée. Chaque membre du groupe a configuré une machine virtuelle Linux Debian sur son ordinateur personnel
Le tutoriel a été pensé et rédigé en équipe via des outils de rédactions collaboratifs
Le rapport d'auto-évaluation était individuel et basé sur une reflexion personnelle sur le travail réalisé
Les productions ainsi regroupée ont permis :
  1. De comprendre le fonctionnement d'une machine virtuelle
  2. D'apprendre a installer et configurer une machine Linux
