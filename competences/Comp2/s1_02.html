Sae 1.02 : Comparaison d'approches algorithmiques

En partant d'un problème posé, l'idée était d'analyser les structures les plus adaptée afin de réduire le temps de résolution d'un problème simple.
Cette SAÉ est la première liée a la compétence 'Optimiser des Applications informatiques'.

Les productions :
  1. Code de tests pour chaques problèmes posés
  2. Rapport d'analyse rédigé des différents problèmes posés ainsi que leur solution
  3. Document d'analyse rédigé sur l'ensemble des types java et leurs impact sur le temps de résolution d'un problème

L’objectif est de pouvoir analyser le fonctionnement et le temps de réponse d'une application, et l'améliorer si besoin


Le sujet proposé etait l'analyse du temps d'execution de problèmes simples dans différentes situation au sein de programme java'
Les différents problèmes posés reposaient sur l'utilisation des types les plus adaptés a la situation


Ce projet a été réalisé en équipe imposée. Chaque programme ont été testé sur les machines de différents membre du groupe
Les temps d'executions ont été claculé à l'aide des méthodes permettant d'accéder a l'horloge système d'une machine
Afin de permettre une analyse convenable, plusieurs essais de temps d'executions ont été réalisé
  1. Vérifier le temps d'éxecution global
  2. Analyser l'optimisation et la vitesse d'execution
  3. Analyser les situations les plus adaptées aux différents types de structures de données Java
  4. Vérifier le fonctionnement sur différentes machines
