Sae 2.04 : Exploitation d'une base de donnees

Cette saé avait pour but de gérer, administrer et optimiser une base de donnée déjà existante
Cette SAÉ fait suite à la saé 'Creation d'une base de donnees'.
Elle détaille les différents moyens de gestion d'une base prédéfinie
Elle prend appui sur la base de donnée de gestion d'un catalogue de base de donnée réalisée durant la Saé précédente

Les productions :
  1. Script de requêtes
  2. Traces d'execution des requêtes
  3. Script des différentes fonctions réalisée
  4. Script de gestion des droits et rôles au sein de la base de données

L’objectif est d'apprendre les différentes possibilitée concernant la gestion d'une base de donnnée


Le sujet proposé etait la création d'une base de donnée permettant de gérer les stocks d'un catalogue de bandes dessinées
Les différents problèmes posés reposaient sur l'organisation des données, ainsi que les différents liens entre chaques tables qui étaient a établir


Ce projet a été réalisé en équipe libre. Chaque membre du groupe a contribuer a l'écriture d'un script ou d'une requête
Les scripts ont été relus et testé par plusieurs membres
Les productions ainsi regroupée ont permis :
  1. De comprendre les méthodes de gestions de permissions d'une base de donnée
  2. D'experimenter la réalisation de requête plpgsql
  3. De comprendre l'intéret de la réalisation de requêtes d'accès typiques
