Sae 1.04 : Creation d'une base de donnees

Cette saé avait pour but de créer une base de donnée de gestion d'un catalogue de Bandes Dessinée
Cette SAÉ est la première liée a la compétence 'Gérer des données de l'information'.
Elle détaille la conception théorique et pratique d'une base de donnée selon un modèle relationnel donné

Les productions :
  1. Dictionnaire de données
  2. Schéma détaillant la structure de la base
  3. Script de création de tables
  4. Script d'insertion de données

L’objectif est d'apprendre les différentes étapes nécessaires a la réalisation d'une base de donnnée


Le sujet proposé etait la création d'une base de donnée permettant de gérer les stocks d'un catalogue de bandes dessinées
Les différents problèmes posés reposaient sur l'organisation des données, ainsi que les différents liens entre chaques tables qui étaient a établir


Ce projet a été réalisé en équipe imposée. Chaque membre du groupe a participer a une étape de la création de la base de donnée
Les scripts ont été relus et testé par plusieurs membres
Les schémas ont été modifié plusieurs fois afin de correspondre au modèle final
Les productions ainsi regroupée ont permis :
  1. De comprendre la réalisation d'une base de donnée
  2. Et Comprendre les différents procédés nécessaires pour créer des liens intelligent entre chaques données
