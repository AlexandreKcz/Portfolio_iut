Sae 2.05 : Gestion d'un projet

Cette saé avait pour but de réaliser une site web relatant l'ensemble des étapes de gestion d'un projet de développement
Cette SAÉ fait suite a la saé 'Recueil de besoins'.
Elle comprend la gestion d'un projet de développement d'une application et la réalisation d'un site web de présentation

Les productions :
  1. Maquettes d'un sites web html relatant la gestion du projet
  2. Oral de présentation du projet

L’objectif est d'apprendre a organiser un projet de développement et apprendre a le présenter a un client


Le sujet proposé etait la réalisation d'une application à destination de Laboratoire pharmaceutique
L'application avait pour but de recueillir une base de données d'analyse d'échantillon
Les différents problèmes posés reposaient sur la gestion du développement, ainsi que les dépendances entre les différentes étapes


Ce projet a été réalisé en équipe imposée. Chaque membre du groupe a contribuer a l'organisation du développement, deux personnes se sont occupées du site
Plusieurs schéma et graphique ont été réalisé selon les méthodes vues en cour de Gestion de Projet

Les productions ainsi regroupée ont permis :
  1. D'expérimenter l'organisation d'un projet à l'échelle professionelle
  2. De comprendre les étapes importantes de la gestion d'un projet
  3. De réaliser une présentation des travaux réalisé a destination d'un client externe
